# 🏥 CarePlus Hospital Website

Modern hospital website built using React.js and Tailwind CSS with a clean, responsive, and user-friendly interface.

---

# 🚀 Features

* Modern Responsive UI
* Clean Component-Based Architecture
* Hero Section
* Services Section
* Emergency Contact Section
* Appointment UI
* Doctor & Hospital Information
* Mobile Friendly Design
* Smooth Layout & Styling

---

# 🛠️ Tech Stack

* React.js
* Tailwind CSS
* JavaScript
* HTML5
* CSS3

---

# 📂 Project Structure

```bash
src/
├── components/
├── pages/
├── assets/
├── App.jsx
└── main.jsx
```

---

# ⚡ Installation

```bash
npm install
npm run dev
```
